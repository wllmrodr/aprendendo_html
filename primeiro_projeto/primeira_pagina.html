Afinal, HTML é uma linguagem de programação ou não?

HTML (Hypertext Markup Language) não é considerado uma linguagem de programação porque não possui a capacidade de realizar operações lógicas ou executar instruções. HTML é uma linguagem de marcação utilizada para estruturar o conteúdo de uma página web, indicando como os elementos devem ser exibidos no navegador.

Estrutura Básica

Para entender como é estruturado, é essencial entender oquê são tags e suas funcionalidades. Uma tag é um elemento utilizado para marcar ou definir partes específicas do conteúdo em uma página web, podendo ser delimitada em dupla, sendo uma de abertura e outra de fechamento ou utilizada sozinha. As tags são fundamentais para a estruturação e formatação do conteúdo HTML, fornecendo instruções ao navegador sobre como exibir os diferentes elementos na página.

Sabendo disso, podemos dizer que a estrutúra básica do arquivo HTML se divide em três pares de tag's: O html que delimita todo o arquivo, a head (cabeça) que denomina o nome da página e o corpo, que contém tudo que vai ser vizualizado pelo usuário.

* É importante notar que a TAG de fechamento sempre tem a presença de uma barra (/) em sua composição.

Exemplos TAG's

Como vimos acima, toda a estrutura do arquivo HTML se baseia no conceitos de tags. Existes numerosas tags com os mais diversos tipos de customização para a página, nesse tópico veremos alguns exemplos:

  • TAG's de Texto:i, strong, input, small, sup, u
  • TAG's de Lista:ul, ol, dl
  • TAG's de Imagem:img
  • TAG's de Formulários:form, label
  • TAG's de Links:a
  • TAG's de Tabela:table, tr, td, th

Atributos de TAG's

Em HTML, um atributo é uma parte adicional de uma tag que fornece informações extras sobre o elemento. Os atributos são usados para modificar ou fornecer detalhes específicos sobre como o elemento deve ser exibido ou comportar-se. Existindo alguns atributos genéricos e outros específicos de cada tag.

Exemplos:

Genéricos

  • strong id = "nome_identificador" -> estabelece um identificador.
  • strong style "color: blue;" -> define uma cor pra determinado texto.
  • strong class = "nome_titulo" -> cria classes com formatos pré definidos.

Específicos

  • input type = "text" ou "number ou "color" -> define formato do input.
  • img src = "endereço_img" -> possibilita a inserção de uma imagem, com referência no endereço oferecido.
  • img width = "250" -> edita largura da imagem inserida.

TAG de Link

A tag que serve pra redirecionamentos e ancoragem de link é (a), normalmente acompanha pelo atributo (href)

Exemplos:

  • a href ="http://site.com" -> redireciona direto pro site de interesse.
  • a href ="http://site.com" target = "_blank" -> abre uma nova aba contendo o site de interesse.
